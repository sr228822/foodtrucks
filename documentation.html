Time Estimates

The Food Trucks endpoint returns information for all food trucks in a given location.

Resource

GET /api/v1/foodtrucks

Query Parameters

Name Type Description
start_lat float Latitude component of start location.
start_lng float Longitude component of start location.
end_lat float Latitude component of end location.
end_lng float Longitude component of end location.
type (optional) string Filter results for a specific type of food truck. Valid values are "Truck" or "Push Cart"
status (optional) string Filter results for a specific application status. Valid values are "APPROVED", "POSTAPPROVED", "REQUEST", "EXPIRED", "ONHOLD"
food (optional) string Filter results that contain this string in their declared food items
perpage (optional) int Number of results per page (default is 50)
page (optional) int Page to request (default is 0)
sort (optional) string How to sort results. Valid values are "locationid" or "alphabetic". Default is "locationid"

Status-Code: 200 OK
